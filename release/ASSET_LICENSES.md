# Asset & Dependency Licenses — Emberwilds

_Last audited: 2026-07-10._

Emberwilds is unusually clean to clear: **there are no third-party art, audio,
or font assets.** Every sprite is a pixel array authored in TypeScript, all
audio is synthesized at runtime via the Web Audio API, and the in-game font is a
hand-drawn bitmap defined in code. The only third-party code is the game engine
(Phaser, MIT) and build tooling.

## Verdict

**Clear for commercial use on all portals.** No attribution is legally required
(Phaser's MIT license asks that its copyright notice be retained, which it is,
inside the distributed bundle). No asset has unknown provenance.

## Visual assets

| Asset | Location | Source | License | Commercial | Attribution |
|---|---|---|---|---|---|
| All character/enemy/tile sprites | `src/gfx/data/*.ts` | Original — pixel arrays authored for Emberwilds | Owned | Yes | No |
| Parallax, particles, terrain, UI art | `src/gfx/*.ts` | Original — drawn procedurally in code | Owned | Yes | No |
| Bitmap font | `src/gfx/text.ts` | Original — hand-authored 4×6 glyphs | Owned | Yes | No |
| App icons | `public/icon-192.png`, `public/icon-512.png` | Original — generated by `scripts/makeIcons.mjs` | Owned | Yes | No |
| Social / OG image | `public/og-image.png` | Original | Owned | Yes | No |
| Marketing screenshots | `store-assets/screenshots/*.png` | Original — captured from the game | Owned | Yes | No |

## Third-party runtime services (not bundled assets, but disclosed for compliance)

| Service | Where | Purpose | Notes |
|---|---|---|---|
| Google Analytics (GA4) | `index.html` gtag | Anonymous funnel analytics | Disclosed in the Privacy Policy. |
| Google AdSense | `index.html`, gated to `emberwilds.fun` only | Ads on the self-hosted build | Not loaded inside portal iframes. |
| Cloudflare Worker | `backend/leaderboard/` | Global leaderboard storage | First-party; stores a device id + chosen name + times. |

On game portals (CrazyGames, Poki, Yandex, GameDistribution) ads and analytics
are the **portal's** SDK, integrated via `src/platform/` — see that layer's
adapters. AdSense stays off any non-`emberwilds.fun` host by design.
