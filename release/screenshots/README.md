# Screenshots

The 10 marketing screenshots (1920×1080 PNG, re-shot on the Phase-3 maps) live in
[`store-assets/screenshots/`](../../store-assets/screenshots/) — not duplicated
here. They cover the title, all six worlds, a boss, the world map, and the
wardrobe.

For a portal that wants everything zipped in one folder, copy them in at
submission time:

```sh
cp ../../store-assets/screenshots/*.png .
```
