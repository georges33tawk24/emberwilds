# Icons

Canonical icons live in `public/` (shipped with the game, referenced by the PWA
manifest). Not duplicated here to keep one source of truth.

- `public/icon-192.png` — 192×192 app icon
- `public/icon-512.png` — 512×512 app icon (maskable-safe padding)
- `public/og-image.png` — 1200×630 social/cover image

Regenerate the app icons with `node scripts/makeIcons.mjs`.
